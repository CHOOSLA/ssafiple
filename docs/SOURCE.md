# 데이터 출처 및 라이선스 정보 (명세 8장 대응)

본 프로젝트는 아래 공공 데이터를 활용하여 구축되었습니다.

## 1. 관광 정보 및 지리 정보 데이터 출처
* **서울시 주요 관광지 정보**: 서울열린데이터광장 및 공공데이터포털
* **공공데이터포털**: [국가중심데이터 - 서울시 관광지 정보 API/JSON]
* **한국관광공사 국문 관광정보 서비스 (국문 API)**: 관광지 좌표, 주소, 이미지 등

## 2. 라이선스 정보
* 본 서비스가 활용한 공공 데이터는 **공공누리(공공저작물 자유이용허락 표시) 제1유형**("출처표시")에 해당합니다.
* 출처를 명시하고 자유롭게 재배포 및 활용할 수 있습니다.
* 데이터 원본 JSON 파일들은 `data/raw/` 디렉토리에 백업 보관하며, 변경 없이 원형 그대로 적재하는 것을 원칙으로 합니다.
