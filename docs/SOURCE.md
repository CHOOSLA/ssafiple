* 본 서비스가 활용한 공공 데이터는 **공공누리(공공저작물 자유이용허락 표시) 제3유형**("출처표시 + 변경금지")에 해당합니다.
* **조건**: 출처 표시 필수, 내용 변경 금지 (형식 변환 및 선별 적재만 허용), 상업적 이용 및 재배포 허용.
* 서비스 하단 푸터에 아래 출처 문안을 상시 표기합니다. (AGENTS.md §6.2와 동일 기준)

* 데이터 원본 JSON 파일들은 `data/raw/` 디렉토리에 백업 보관하며, 변경 없이 원형 그대로 적재하는 것을 원칙으로 합니다.
